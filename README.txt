

KATOPODIS PANAGIOTIS ARISTOMENIS 


Sthn arxh tou arxeiou iparxei ena sxediagramma me ta index block kai datablock pou apoteloun to dentro.Epishs iparxei se kathe arxeio to block to opoio einai to info block kai periexei plhrofories gia ta type kai length pou periexontai se kathe leaf tou dentrou.

Apo tis sinartiseis pou zitountai ilopoithikan oles ektos apo thn CloseIndex kai CloseIndexScan oi opoies omos exoun kodika mesa.Gia na trexei h main prepei na sxoliastoun aftes.Sto telos tou Insertentry  iparxei min print h opoia ektiponei to dentro gia kathe arxeio oste na diapistosoume an dimiourghthike sosta omos epeidh einia kathara gia testing kai den zititai einai sxoliasmenh sto telos ths instert kai an theloume na treksei prepei na valoume manual ton arithmo ton egrafoon pou ginetai insert opos sth main1 gia paradeigma einai 100.Mesa sthn OpenIndexScan iparxei sxoliasmenh mia megalh switch h opoioa ilopoioei ta erothmata Query h opoia xrisimopoihthike kai auth gia testing.

Extra sinartiseis pou dimiourgithikan:
1)push stack : kanei push to stoixeio sto stack
2)pop staxk : kanei pop to stoixeio apo to stack
3)print : ektiponei analoga me to type tou value
4)printAll : ektiponei olokliro to dentro
5)compareTypes : sigrinei ta values kai xrisimeuei giati mporei kai sigrinei void* kanontas cast
6)compareTypesEqual : opos kai apo pano apla elenxei kirios gia ison h mh ison.
7)checkFull : elenxei an ena block einai gemato kai an xoraei allh eggrafh
8)returnOffset : epistrefei to offset mias eggrafhs pou vrisketai mesa se ena block 
9)returnOffsetScan : to idio me thn apo pano me th diafora pos auth epistrefei to proth thesi pou tha brei to stoixeio kai tha agnohsei ama einai idia stoixeia os epomena
10)searchRoot : ths dinoume mia egrrafh void*value kai afti mas epistrefei to blocknum sto opoio iparxei h eggrafh
11)inserIndexBlock : auth h insert diaxeirizetai thn eisagogh timis sto eurethrio
